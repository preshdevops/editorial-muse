# 💌 Editorial Muse

> *Because some feelings deserve more than a text message.*

**Editorial Muse** is a beautifully crafted letter-writing web app that helps you compose heartfelt, personal letters for the people who matter most — and send them straight to their inbox with a single click.

Whether it's a love letter, a thank-you note, a letter to an old friend, or words you've been meaning to say for years — Editorial Muse gives those words the elegance they deserve.

---

## ✨ Features

- **✍️ Write with intention** — A clean, distraction-free writing experience designed to help you pour your heart onto the page
- **💌 Letter-first design** — Your words are styled beautifully, like a real letter — not a boring email form
- **📬 Send via email** — Deliver your letter directly to your loved one's inbox, no extra apps needed
- **🎨 Warm, personal aesthetic** — Every detail of the UI is crafted to feel intimate and thoughtful
- **⚡ Fast & lightweight** — Built with plain HTML & JavaScript, so it loads instantly anywhere

## 🚀 Getting Started

Want to run it locally? It's simple:

```bash
# Clone the repository
git clone https://github.com/preshdevops/editorial-muse.git

# Navigate into the project folder
cd editorial-muse/editorial-muse

# Open in your browser
open index.html
```

No installs. No build steps. Just open and write.
